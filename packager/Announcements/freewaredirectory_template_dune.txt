%%name: white_dune

%%short-description: graphical X3DV/VRML97 editor/animationtool

%%full-description: white_dune is a graphical editor, a simple NURBS 
                    3D modeller, a animation tool and a compiler
                    for the Extensible 3D (X3D(v): ISO/IEC 19776-2.2:2008) 
                    and Virtual Reality Modelling Language 
                    (VRML97: ISO/IEC 14772) in development.
                    X3D and VRML97 are the ISO standards for displaying 
                    3D data over the web via browserplugins.
                    X3D and VRML97 have support for animation, realtime 
                    interaction and multimedia (image, movie, sound).

                    white_dune can read X3DV/VRML97 files, display and let the user 
                    change the scenegraph/fields. 

                    white_dune has support for stereoscopic view via 
                    "quadbuffer" capable stereo visuals. 

                    white_dune can export the data of the X3D scenegraph
                    to sourcecode in the languages C, C++ und java
